# NeoVim Configuration
personal NeoVim configuration written in lua.

- fugitive
- harpoon
- lsp
- telescope
- treesitter
- undotree

## Setup
### 1. Clone packer
Clone [Packer Repo](https://github.com/wbthomason/packer.nvim) in nvim config directory

### 2. Install packages
Navigate to packer.lua and run:
  - `:so`
  - `:PackerSync`
